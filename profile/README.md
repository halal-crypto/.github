## Добавляем jup.ag 25h
- апи не даёт выгрузить список живых токенов - придётся либо парсить с сайта, либо спарсить один раз и зашить в конфиг сборщика
- на сайте нашёл только 5 тикеров к USDT и 1 к USDC - оно того стоит вообще?
- в апи используются не человеческие имена тикеров, а внутренние идишки вида `CcVV4V6572y4bp1Ca5z5o8yr2kd36CzNY9HfTMZ3LzdV`
- в апи нет данных по объёму, вводу выводу
- цена, видимо, приводится к USD. Символы скорее вида `PUMP/USDT/USD` vs `PUMP/USDT`
- судя по вводным выше - придётся знатно полюбиться, чтобы это добавить в нашу унифицированную систему учёта. Меньше 20-30 часов я бы точно не брал.
